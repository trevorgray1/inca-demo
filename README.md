# Blockchain Data ETL Pipeline

This project extracts, processes, and stores blockchain data using Python.

## Structure
- `etl/`: ETL logic (fetch, process, load)
- `config/`: Configuration files
- `scripts/`: Entry-point scripts

## Getting Started
1. Install dependencies: `pip install -r requirements.txt`
2. Add your API key and DB info to `config/config.yaml`
3. Run the pipeline: `python scripts/run_etl.py`
